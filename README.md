# Erno

![Status: Alpha](https://img.shields.io/badge/status-alpha-blue)

Simple visual tool to create Rubik's Cube patterns from a custom or scrambled state.

Live page: https://yaacoub.github.io/apps/erno

## Description

Erno helps users move from one cube state to another with clear algorithm output and a step-by-step view.

This project is currently in alpha and serves as an MVP. For this phase, development is intentionally fast and includes AI-assisted code generation. The codebase will be refined progressively as the project evolves.

## Getting Started

### Dependencies

- Node.js 24+
- Modern browser

### Installing

```bash
git clone https://github.com/yaacoub/Erno.git
cd Erno
npm install
```

### Executing program

```bash
# Start development server
npm run dev

# Build production bundle
npm run build

# Preview production build
npm run preview
```

Open http://localhost:5173 in your browser when running in dev mode.
